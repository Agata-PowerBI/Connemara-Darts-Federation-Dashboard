# Connemara-Darts-Federation-Dashboard

1.	Project Title:
   
Connemara Darts Federation Season Summary – Season 1 and 2

2.	Goal:
   
Presenting overall results’ summary among all Venues and Players

3.	Tools and Technologies used:
   
Power BI Desktop - Microsoft BI tool for Data Visualisation
Power Query - data transformation and cleaning layer for reshaping and preparing the data
DAX language - used for calculated measures, dynamic visuals and conditional logic
Data Modelling- establishing relationships between tables to enable interactions, cross-filtering and aggregation
File format .pbix for development and .png for dashboard previews

4.	Data source 
Report is connected to Excel file

5.	Features / highlights
Project was built at the request of CDF secretary.
 
Report consists of the following visuals:
Cards – to present results sums, averages and max KPIs
Clustered Bar Charts – to present Top Venues and Players by 180 score
Slicers – to enable filter the results by Venue, Division and Player
